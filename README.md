# go-diff
Building own diff command line tool.

## Run command
```
go run cmd/bin/main.go <original-file-path> <new-file-path>
```

## Build and Run Command
```
go build cmd/bin/main.go
./main <original-file-path> <new-file-path>
```

## Run Tests
```
go test ./tools
```
